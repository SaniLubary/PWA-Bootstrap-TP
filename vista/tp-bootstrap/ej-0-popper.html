¿Necesito popper.js para Bootstrap? ¿por qué? ¿para qué?

No se necesita, es una libreria independiente a Bootstrap.

Popper.js provee utilidad de posicionamiento dinámico y detección de viewport, para así mostrar info extra en formato popover sobre otros elementos de la página.

Esta se incluye en la versión bundle de bootstrap, por si quieres hacer uso de ciertas funcionalidades, como por ejemplo el uso de componentes Dropdown o Popover lo requieren.

Aunque si tienes la version min.js de Bootstrap, tendras que agregar Popper.js por separado, o usar la versión bundle de Bootstrap
